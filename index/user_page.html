Stels Navigator

Горный велосипед для начинающих с навесным оборудованием Shimano, 18 скоростей. Технические особенности: прочная стальная рама, амортизационная вилка XDS с ходом 60 мм, двойные алюминиевые обода, надежные ободные тормоза V-brake, подножка. Подходит для прогулочного катания по различным дорогам и пересеченной местности

Стоимость проката: 100 руб. в час

Stinger Defender

Подходит для прогулочного катания по различным дорогам и несложной пересеченной местности.
